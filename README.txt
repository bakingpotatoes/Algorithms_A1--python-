1. Install the files as zip
2. Extract as a folder of any type
3. Open your VSC and select that folder
4. Open your cmd in VSC
5. Ensure that pip is installed in your current interpreter's folder
6. Type "pip install -r requirements.txt" (dont worry its just pygame)
7. In the bottom right, select the interpreter that step 6 installed pygame into (very important to have pip and pygame in the same folder)
8. The "import" section should become NOT white
9. Make sure you're on the main.py and select it as the file to debug (with whatever method of running the file you prefer, F5 or the play button)
